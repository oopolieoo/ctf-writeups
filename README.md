# ctf-writeups

This repository contains write-ups for permitted Capture the Flag (CTF) challenges. The focus is on methodology, exploitation logic, lessons learned, and the reasoning behind each step.

## Structure
Write ups are organized by platform and challenge type.

## Discolsure
All CTF's posted to this repository follow platform rules. No active challenges, credetials, sensitive information, or flags are included. Identifying details will be redacted accordingly.
